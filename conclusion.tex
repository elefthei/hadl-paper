\section{Conclusion}
\label{sec:conclusion}

We have presented \hadl, a typed imperative DSL for long-running agentic workflows that treats agentic plans as programs with \texttt{ask}/\texttt{say}/\texttt{gen}/\texttt{agent} as first-class typed expressions. The central contribution is \emph{gradual validation}, which unifies three LLM-generation patterns (types via \texttt{Schema}, workflows via arrow types, Cedar authorization via \texttt{Policy}) under a single re-checking-plus-bounded-retry discipline, with per-step soundness mechanized in Lean~4. Append-only policy merging plus Cedar's \texttt{forbid}-over-\texttt{permit} precedence ensure dynamically generated policies can only tighten access. Remaining work includes mechanizing the trace-level termination measure, parallel \texttt{gen} call expressions for multi-agent fan-out, extending the type language with richer record and sum types, and support for workflow imports to scale to larger systems.
