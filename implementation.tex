\section{Implementation}
\label{sec:implementation}

\pact is implemented in Rust: PEG parser, type checker, and async tree-walking interpreter with no code generation. \texttt{js} expressions and backtick interpolation run on an embedded JavaScript engine (Boa) with \pact variables mirrored as globals. \texttt{gen}/\texttt{agent} calls are dispatched through the GitHub Copilot SDK. \texttt{ask}/\texttt{say} bind to a configurable channel. \texttt{eval} deserializes a JSON-serialized workflow produced by an earlier \texttt{gen} and re-enters the interpreter in the caller's environment. Before each \texttt{gen}/\texttt{agent} call the Cedar check of \sect{sec:design-gradual} runs under the principal given by the \texttt{as} clause, halting the workflow on deny. The staged re-checker and policy-install invariants of \sect{sec:design-gradual} are implemented directly by the interpreter, backed by a central policy store seeded by the \texttt{@policy} pragma. The full source is in the submission artifact.
